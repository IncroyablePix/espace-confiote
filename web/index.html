Available Functions:

  • player.shoot(direction) - Fire a projectile
    direction: int - 0 (forward), 1 (left), 2 (right)
  • player.set_thrusters(left_power, right_power) - Control movement
    left_power, right_power: float - Range: -1.0 to 1.0
  • print(message) - Output to browser console

Context Object (passed to on_player_update):

  • context["playerPosition"] - dict with "x" and "y"
  • context["playerVelocity"] - dict with "x" and "y"
  • context["playerRotation"] - float - Rotation angle in degrees
  • context["deltaTime"] - float - Time since last frame (seconds)
  • context["visibleMonsterCount"] - int - Number of visible monsters
  • context["visibleMonsters"] - list - Array of monster objects

Monster Object Properties:

  • monster["position"] - dict with "x" and "y"
  • monster["size"] - float - Monster size/radius
  • monster["health"] - float - Current health (0-100)
  • monster["invincible"] - bool - Whether monster is invincible
